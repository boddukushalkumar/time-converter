📚 Understand the Concepts

Let's learn the difference between 12-hour and 24-hour time!

🕐

What is 12-Hour Time?

The 12-hour clock divides the day into two halves:

  • AM Midnight to Noon (morning)
  • PM Noon to Midnight (afternoon/evening)
7:30 AM = Morning
4:45 PM = Afternoon
🕛

What is 24-Hour Time?

The 24-hour clock counts hours from 00 to 23. There is no AM or PM!

07:30 = 7:30 AM
16:45 = 4:45 PM
23:59 = 11:59 PM
00:00 = Midnight
📏

Conversion Rules

AM Rules

✔ 12:00 AM → 00:00
✔ 1:00 AM → 01:00
✔ 9:00 AM → 09:00
✔ 11:00 AM → 11:00

PM Rules

✔ 12:00 PM → 12:00
✔ 1:00 PM → 13:00
✔ 2:00 PM → 14:00
✔ 6:00 PM → 18:00
✔ 11:00 PM → 23:00

💻 Python Code Explained

Click each line to see a friendly explanation!

time = input("Enter time (HH:MM:SS AM/PM): ")
💡 Reads the time entered by the user from the keyboard. For example: 02:30:45 PM
parts = time.split()
💡 Separates the clock part and AM/PM using the space. Result: ["02:30:45", "PM"]
clock = parts[0]
💡 Stores the time part. Gets the first item: "02:30:45"
period = parts[1]
💡 Stores AM or PM. Gets the second item: "PM"
hour, minute, second = clock.split(":")
💡 Splits the time by the colon : into three parts: hour="02", minute="30", second="45"
hour = int(hour)
💡 Converts the hour from a text string into a number so we can do math with it. "02" → 2
if period.upper() == "AM":
💡 Checks if the time is in the morning (AM). The .upper() makes sure both "am" and "AM" work!
if hour == 12:
💡 Special case! 12 AM is midnight, which is 00:00 in 24-hour time. So we set hour to 0.
hour = 0
💡 12 AM becomes 00 (midnight). This is the trickiest part of time conversion!
else: # PM
💡 If it's not AM, then it must be PM (afternoon/evening). Different rules apply!
if hour != 12:
💡 For PM times, we add 12 to the hour — except 12 PM, which stays as 12. So we check if hour is NOT 12.
hour += 12
💡 Adds 12 to convert PM hours. Example: 2 PM → 2+12 = 14. 6 PM → 6+12 = 18.
print(f"{hour:02}:{minute}:{second}")
💡 Displays the final 24-hour time! The :02 makes sure the hour always has two digits (e.g., 09 not 9).

🎉 Fun Facts

Did you know? 24-hour time is used all around the world!

🌍

Most countries around the world use 24-hour time as their standard.

✈️

Airports use 24-hour time on departure boards to avoid AM/PM confusion.

🚂

Railways and train schedules worldwide use 24-hour time for clarity.

🏥

Hospitals use 24-hour time to prevent medication errors. Safety first!

🚢

The Navy and military use "military time" — that's 24-hour time!

🖥️

Computers and servers internally use 24-hour time (and often UTC).
